# Brazil Education and GDP Dataset (2001–2020)

This repository contains a cleaned dataset comparing primary school enrollment and GDP per capita for Brazil between 2001 and 2020.

## Files
- `brazil_education_gdp.csv` — Final cleaned dataset.
- `brazil_education_gdp.db` — SQLite database version of the dataset.
- `data_description.json` — Metadata describing each field.

## Data Sources
- World Bank Education Statistics (API_SE.PRM.ENRL_DS2)
- World Development Indicators (WDI)
